# User Management System (ASP.NET Core)

A full stack user management system built with **ASP.NET Core** featuring **authentication**, **role-based authorization**, and an **admin dashboard** to manage users and roles.

## Features
- User registration & login
- Role-based access (Admin / User)
- Admin dashboard:
  - View users
  - Create / edit / deactivate users
  - Assign roles
- Database integration (SQL) with CRUD operations
- Form validation & error handling
- Responsive UI using Bootstrap 5

## Tech Stack
- ASP.NET Core
- Entity Framework Core
- ASP.NET Core Identity
- SQL Server
- Bootstrap 5.2
- Git / GitHub
